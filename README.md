# The Vibe Codex — AI Coding Cheat Sheet

> The Vibe Codex is an AI coding cheat sheet designed specifically for vibe coders.

## Overview

The Vibe Codex serves as a comprehensive cheat sheet and reference guide for developers who embrace "vibe coding"—an intuitive, AI-assisted approach to software development. This project aims to streamline the workflow of modern developers by providing quick access to essential prompts, patterns, and practices used when collaborating with AI coding assistants.

By centralizing these resources, The Vibe Codex solves the problem of fragmented knowledge in the rapidly evolving field of AI-assisted development. It offers a clean, accessible interface built with modern web technologies, ensuring that developers can quickly find the information they need to maintain their coding momentum and creative flow.

## Features

| Feature | Description |

## Setup & Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/reginaldcobb/thevibecodex.git
   cd thevibecodex
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
5. Run the production build:
   ```bash
   npm start
   ```

## Usage

Once the development server is running, navigate to the provided local URL in your web browser. Browse through the various categories of the cheat sheet to find relevant AI coding prompts and patterns. Use the copy buttons to quickly grab snippets for use in your AI coding assistant of choice.
